# CreatorCalc (เครื่องคำนวณรายได้ครีเอเตอร์)

เครื่องมือวิเคราะห์และวางแผนการเงินระดับพรีเมียมสำหรับครีเอเตอร์มืออาชีพ ช่วยให้คุณมองเห็นภาพรวมรายได้จาก YouTube และวางแผนการจัดการเงินอย่างมีระบบตามกฎ 10-5-85

## คุณสมบัติเด่น

- **การคำนวณรายได้ที่แม่นยำ**: คำนวณรายได้รายวัน รายเดือน และรายปี ตามจำนวนวิดีโอ ยอดวิว และค่า RPM
- **รองรับสองสกุลเงิน**: สลับระหว่าง USD และ THB ได้อย่างราบรื่นพร้อมคำนวณอัตราแลกเปลี่ยนปัจจุบัน (1 USD = 31.59 THB)
- **กฎการจัดสรรเงิน 10-5-85**:
    - **10% เงินเดือน (Executive Salary)**: สำหรับค่าใช้จ่ายส่วนตัวและการใช้ชีวิต
    - **5% ระบบ (System Growth)**: สำหรับการลงทุนในอุปกรณ์ ซอฟต์แวร์ และการพัฒนาคุณภาพงาน
    - **85% เครื่องยนต์สร้างความมั่งคั่ง (Wealth Engine)**: ลงทุนใน S&P 500 เพื่ออิสรภาพทางการเงินในระยะยาว
- **การพยากรณ์ความมั่งคั่งด้วยกฎ 4%**: วิเคราะห์การเติบโตของเงินลงทุนใน S&P 500 (ผลตอบแทน 7% ต่อปี) พร้อมแสดงรายได้คงที่ (Passive Income) ที่คุณจะได้รับโดยไม่กระทบเงินต้น
- **ดีไซน์ระดับพรีเมียม**: อินเตอร์เฟซแบบ Glassmorphism ที่สวยงาม ทันสมัย รองรับทั้งโหมดมืด (Dark Mode) และโหมดสว่าง (Light Mode)
- **รองรับการใช้งานบนมือถือ**: สามารถเพิ่มไอคอนบนหน้าจอหลัก (Add to Home Screen) เพื่อเข้าใช้งานได้เหมือนแอปพลิเคชันปกติ
